## Taking Screenshots

### On macOS:
- Press `Cmd + Shift + 4` to take a screenshot of a selected area
- Press `Cmd + Shift + 3` for full screen
- Use `Cmd + Shift + 4` then `Space` to capture a window

### Tips:
- Use a clean browser window (hide bookmarks bar if possible)
- Show meaningful data in visualizations
- Capture at a good resolution (at least 1920x1080 or higher)
- Consider using browser dev tools to set a specific viewport size for consistency
- Make sure the UI looks polished (no console errors visible)

## File Naming Convention

Use descriptive, lowercase names with hyphens:
- `main-interface.png`
- `query-results.png`
- `dark-mode.png`
- `export-menu.png`
